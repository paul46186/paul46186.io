    Assembly line balancing problem with machine and task constraints - The case study of a shoe manufacturing company

    The primary goal of this project is to create a model that can optimize the assembly line layout in a matter of seconds. We initially cleaned the data and made it readable with Excel VBA. The math equations are then constructed to fit all of the requirements. We coded in Python and used Gurobi to optimize the result of this project. We split our model into two pieces because the global optimization will take a day to complete, which is not ideal in practice. We were able to get the heuristic findings in seconds using the new two models. Furthermore, the result is similar to the global optimization.
